# Morel

**Deep luxury.** Near-black surfaces with champagne gold and forest green. For products where trust and quality are the core value proposition.

**Voice:** Measured, authoritative, considered. "Built for those who care about the details."

**Best fit:** Fintech, premium SaaS, enterprise.

## Typography

| Role | Font | Weight | Size | Leading |
|---|---|---|---|---|
| Display | Playfair Display | 700 | 48px | 1.08 |
| H1 | Playfair Display | 700 | 34px | 1.18 |
| H2 | Playfair Display | 600 | 24px | 1.3 |
| H3 | Outfit | 500 | 17px | 1.4 |
| Body | Outfit | 400 | 15px | 1.7 |
| Caption | Outfit | 300 | 12px | 1.55 |
| Label | Outfit | 600 | 10px | 1.4 |
| Code | Fira Code | 400 | 13px | 1.6 |

**Rationale:** Playfair Display delivers unmistakable editorial luxury -- its high-contrast strokes catch light on the dark surface. Outfit is a round-geometric sans that feels modern without competing. The size differential between display and body is wide by design.

## Signature Detail

A gold gradient border (`linear-gradient(135deg, rgba(184,154,94,0.4), rgba(184,154,94,0.08))`) on the top and left edges of cards only -- right and bottom edges have no border. Creates a directional light source illusion.
