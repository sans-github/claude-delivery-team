# Veldt

**Cool restraint.** Near-monochrome with a single electric blue accent. For developer tools, data products, and infrastructure software.

**Voice:** Precise, direct, confident. "Complexity is handled here."

**Best fit:** Developer tools, infrastructure, data platforms.

## Color Palette

| Token | Hex | Role |
|---|---|---|
| Primary | `#0057FF` | CTAs, links, active states |
| Primary Hover | `#0041CC` | Hover on primary |
| Primary Subtle | `#E5EEFF` | Tinted backgrounds, badges |
| Secondary | `#3A4B5C` | Supporting actions |
| Background | `#F2F3F5` | Page background |
| Surface | `#FFFFFF` | Cards, panels |
| Text | `#0D1117` | Headings, body |
| Text Secondary | `#5A6472` | Labels, secondary |
| Border | `#D4D8DE` | Dividers |
| Success | `#006644` | Confirmations |
| Error | `#CC1616` | Errors |

## Typography

| Role | Font | Weight | Size | Leading |
|---|---|---|---|---|
| Display | Syne | 700 | 56px | 1.0 |
| H1 | Syne | 700 | 38px | 1.1 |
| H2 | Syne | 600 | 24px | 1.2 |
| H3 | Syne | 600 | 17px | 1.3 |
| Body | Source Serif 4 | 400 | 16px | 1.7 |
| Caption | Source Serif 4 | 400 | 13px | 1.55 |
| Label | Syne | 700 | 11px | 1.4 |
| Code | JetBrains Mono | 400 | 13px | 1.6 |

**Rationale:** Syne is a geometric sans with unusual letterforms that signal technical sophistication. Source Serif 4 in body copy is unexpected for a dev-tool brand and creates distinction. The pairing reads as expert, not sterile.

## Signature Detail

A 3px vertical line in `#0057FF` flush to the left edge of informational panels, no border-radius on that edge. Reads like margin annotations in technical documentation.
